---
description: How to restart the StudyFlow project after a shutdown
---

Follow these steps to get StudyFlow running again:

1. **Terminal 1 (Backend)**
   - Run: `cd backend`
   - Run: `npm run dev`
   - *Result*: Should see "StudyFlow Server running on port 5000" and "MongoDB Connected"

2. **Terminal 2 (Frontend)**
   - Run: `cd frontend`
   - Run: `npm run dev`
   - *Result*: Should see a Local URL, usually `http://localhost:5173`

3. **Browse**
   - Open your browser to the Local URL provided by the frontend terminal.
